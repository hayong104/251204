Vite+Netlify 환경에서 개발 중인 웹 애플리케이션이야
ChatGPT API를 활용하는 간단한 챗봇 기능을 만들어줘.
학생들의 활동에서 잘못된 부분(길이, 각도, 평행 등)을 알려주는 챗봇이면 좋겠어.
API Key는 환경변수(.env 파일)에 VITE_OPENAI_API_KEY로 저장되어 있어
서버 함수는 별도로 사용하지 않고, JS 코드에서 환경변수(.env)에 저장된 API Key를 직접 불러와서 사용할거야
@src/style.css @src/main.js@index.html 에 나눠서 코드를 작성해줘
최상단에 내 API Key가 정상 작동하는지 표시해주는 기능을 만들어줘
​눈이 편안하게 녹색 계열의 파스텔톤 스타일로 만들어줘
